# LRBA Interactive Deck (GitHub Pages)

Este proyecto es una presentación interactiva (tipo “slides”) hecha con **Reveal.js** + **Mermaid**.

## Cómo publicarlo en GitHub Pages (rápido)

1. Crea un repo (ej. `lrba-deck`).
2. Sube TODO el contenido de esta carpeta (mantén `index.html` en la raíz).
3. En GitHub:
   - **Settings → Pages**
   - Source: **Deploy from a branch**
   - Branch: **main** / folder: **/** (root)
4. Abre la URL que te da GitHub Pages.

## Controles
- Flechas: navegar
- `F`: fullscreen
- `S`: speaker notes
- `Esc`: overview

## Notas
- Mermaid y Reveal usan CDN; requiere internet.
- Si deseas “modo offline”, descarga las librerías y colócalas en `/vendor` (te lo preparo si lo pides).
